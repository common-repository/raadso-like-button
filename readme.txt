== Installation ==

1. Upload `raadso-like-button.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

= Will Raadso Like Button Support Other Social Sites? =

Currently Raadso Like Button can only support Raadso Social.
